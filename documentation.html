Qu'est-ce que la stéganographie ?

La stéganographie est l'art de dissimuler des informations dans un support apparemment anodin. Contrairement au chiffrement qui rend un message illisible, la stéganographie cache l'existence même du message.

Exemple simple : Imaginez que vous écriviez une lettre et que vous utilisiez certains mots en gras pour former un message secret. Le message principal reste lisible, mais cache un second message.

Stéganographie d'Image

La méthode LSB (Least Significant Bit)

Cette technique modifie le bit le moins significatif des pixels d'une image pour y cacher des données. Ces modifications sont imperceptibles à l'œil nu.

Avant modification

Pixel: 11101010

Le dernier bit sera modifié

Après modification

Pixel: 11101011

Le dernier bit contient maintenant une donnée cachée

Conseils pour de meilleurs résultats :

  • Utilisez des images de haute qualité
  • Préférez les formats sans perte (PNG)
  • Évitez les images très uniformes
  • Ne modifiez pas l'image après l'encodage

Stéganographie Audio

Principales Techniques

1. LSB Audio

Similaire à la technique LSB des images, mais appliquée aux échantillons audio. Les modifications sont imperceptibles à l'oreille humaine.

2. Echo Hiding

Introduit de petits échos dans le signal audio. Les variations dans le délai et l'amplitude de ces échos permettent d'encoder des données.

3. Phase Coding

Modifie la phase du signal audio initial pour encoder l'information, tout en maintenant les relations de phase entre les canaux.

Considérations importantes :

  • La qualité du fichier audio influence la capacité de stockage
  • Les formats compressés (MP3) peuvent détruire les données cachées
  • Le bruit de fond peut aider à masquer les modifications

Stéganographie de Texte

Méthodes de Dissimulation

Espaces Blancs

Utilise différentes combinaisons d'espaces, tabulations et retours à la ligne pour encoder l'information.

Caractères Unicode

Exploite les variations de caractères et les caractères invisibles pour cacher des données.

Exemple pratique :

Le texte "Bonjour" peut contenir des espaces doubles après certains mots pour encoder des bits (1 pour double espace, 0 pour espace simple).

Stéganographie PDF

Techniques Spécifiques aux PDF

Métadonnées

Les PDF contiennent des champs de métadonnées qui peuvent être utilisés pour stocker des informations de manière discrète.

Annotations Cachées

Les PDF permettent d'ajouter des annotations invisibles qui peuvent contenir des données cachées.

Sécurité et Bonnes Pratiques

Points importants à retenir :

  • La stéganographie seule n'est pas un moyen de chiffrement
  • Combinez toujours avec un chiffrement fort pour les données sensibles
  • Évitez de réutiliser les mêmes fichiers porteurs
  • Gardez privés vos fichiers porteurs originaux

Recommandations de Sécurité

  • Utilisez le chiffrement AES proposé dans les options
  • Choisissez des mots de passe forts et uniques
  • Variez les méthodes et les supports utilisés
  • Effacez les fichiers temporaires après utilisation
